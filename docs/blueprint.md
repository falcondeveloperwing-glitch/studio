# **App Name**: ReplyRush AI

## Core Features:

- AI Conversation Assistant: Utilize AI to analyze Instagram DMs, generate context-aware reply suggestions, and enable automated responses to common customer inquiries to assist in sales closures.
- Modern DM Inbox: Provide a centralized and intuitive inbox for managing live Instagram DM conversations, featuring customer chat lists, message status indicators, and customer labeling.
- AI Knowledge Base Builder: A dedicated tool allowing businesses to train the AI with their specific FAQs, product pricing, delivery information, and store policies, with all data securely stored in Firestore.
- Visual Automation Workflow Creator: Enable users to visually construct custom automation rules using a drag-and-drop interface with connected workflow cards and lines (e.g., 'When customer asks price → send product info').
- Performance Analytics Dashboard: Display key sales metrics, conversation statistics, and AI performance scores through animated charts and a live activity feed for actionable insights, backed by Firestore data.
- Secure User Authentication: Implement robust user registration, login, and profile management functionalities using Firebase Authentication to ensure secure access to the platform.
- SaaS Settings & Integrations: Manage business profiles, connect Instagram accounts, configure AI tone, adjust notification settings, and handle team members, with preferences stored in Firestore.

## Style Guidelines:

- Primary color: Deep violet (#6814F7), selected for its vibrant and electric quality, echoing a futuristic and luxurious feel against dark backgrounds.
- Background color: A very dark, muted purple (#17141C), providing a sophisticated and subtle base that perfectly complements the primary color within a dark UI.
- Accent color: A lighter, ethereal blue (#A1CBFF), offering energetic highlights and creating glowing contrast, evocative of digital luminescence.
- Headline font: 'Space Grotesk' (sans-serif), chosen for its modern and tech-forward aesthetic that suits a futuristic brand. Body font: 'Inter' (sans-serif), ensuring exceptional readability and a clean, neutral appearance for all textual content.
- Minimalist, geometric line-art icons that align with the app's modern, luxury SaaS aesthetic and clean interface.
- Clean, spacious, and fully responsive layouts, emphasizing clarity and sophistication, with UI elements presented in elegant Glassmorphism cards.
- Smooth, high-fidelity micro-animations and glowing hover effects on interactive elements, coupled with smooth transitions, animated loading states, and elegant loading skeletons to provide a truly polished and premium user experience.
